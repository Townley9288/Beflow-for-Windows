# Beflow for Windows v1.1.0.3

本次版本修复部分旧番剧在信息解析阶段提示“找不到可执行的 ffmpeg 文件”、导致无法解析和下载的问题。

## FFmpeg 解析修复

- 所有 BBDown 信息解析命令现在都会显式使用 Beflow 随附的 FFmpeg，不再依赖 BBDown 当前工作目录或系统 PATH。
- 修复旧番剧、整季分析、逐集规格确认和音频选流等流程可能因找不到 FFmpeg 而失败的问题。
- 使用 `ss4232` 完成真实解析验证，可正确识别全部 12 集及各集视频、音频规格。
- FFmpeg 路径缺失时仍保留原有错误诊断和任务日志，便于确认安装文件是否完整。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
