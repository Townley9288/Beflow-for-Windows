# Beflow for Windows v1.0.3

本次版本新增原生主题切换，并改进在线更新检查的稳定性。

## 新功能

- 侧边栏底部新增主题按钮，可以选择“跟随系统、浅色、深色”。
- 主题选择会立即保存，安装版和便携版重启后均会自动恢复。
- 深色模式同步应用于标题栏、侧边栏、页面、卡片、输入控件和日志区域。

## 修复与改进

- 为深色模式增加不透明的专用窗口背景，修复侧边栏文字对比度不足和界面灰蒙的问题。
- 在线更新不再调用 `api.github.com`，避免公共 GitHub API 配额造成频繁检查失败。
- 更新检查通过 GitHub Releases 最新稳定版跳转获取版本，并从对应标签读取 `RELEASE_NOTES.md`。
- 五分钟内重复检查使用内存缓存，仍支持自动下载、SHA-256 校验和安装。
- 关于页增加 Beflow 项目主页与 Releases 链接，并在没有发布日期时隐藏无效日期。

安装版覆盖升级和便携版更新都会保留现有配置、主题选择、历史、任务日志及 B 站登录数据。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
