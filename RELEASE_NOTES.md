# Beflow for Windows v1.1.0.5

本次版本新增 B 站链接剪贴板监听与拖放输入，统一为全部分集解析流程，并优化影视重命名结果展示和逐集编码选择的稳定性。

## 便捷输入与解析

- 运行期间可监听剪贴板中的 `bilibili.com`、`b23.tv` 链接以及 BV、AV、EP、SS 编号；识别后自动恢复并激活 Beflow，进入下载页解析全部分集，但不会自动下载。
- 严格校验 B 站域名和内容编号，支持分享文字与带查询参数的视频、番剧链接，同时拒绝伪装域名、空间主页及普通剪贴板文本。
- 当前有下载、登录或封装任务时暂存最新链接，待任务结束后再自动解析，避免覆盖正在运行的工作区。
- 下载页支持拖入单个链接；多音轨页可分别拖入来源 A/B，一次拖入两个链接时自动切换为双链接模式并同时填充。
- 设置页新增“监听剪贴板”和“监听拖拽链接”两个独立开关，切换后即时生效并自动保存。
- 下载与多音轨页面统一为“开始解析”，始终解析全部分集；下载规格选择与开始下载仍由用户确认。

## 下载与多音轨稳定性

- 逐集编码选项改为值绑定的可观察选项，修复批量规则切换编码后界面选择与实际下载规格可能不同步的问题。
- 修复多音轨逐集自定义延迟开关的单向状态绑定，确保延迟输入框随开关可靠启用或禁用。
- 拖放输入和剪贴板输入共用同一套链接提取与校验逻辑，避免不同入口识别结果不一致。

## 影视重命名体验

- TMDB 匹配成功后只在基本信息卡片显示完整匹配状态，不再重复显示顶部成功通知。
- 执行重命名后，右侧区域从“重命名预览”切换为“重命名结果”，保留原文件名与最终文件名对照以及操作数量。
- 最终文件名支持自动换行与完整悬停提示；左侧源文件列表保持紧凑，便于继续浏览当前目录。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。新安装及旧配置升级后，两个便捷输入开关默认开启，可随时在设置页关闭。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
