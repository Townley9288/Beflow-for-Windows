# Beflow for Windows v1.1.0.1

本次版本在解析式下载基础上加入可自定义的下载目录与文件名规则，并整合下载、重命名历史，进一步完善批量任务的信息展示与响应式界面。

## 下载命名

- 新增“命名规则”页面，分别保存“单集视频”和“多集内容”两套下载规则。
- 可独立配置视频主文件夹、可选子文件夹和文件名，不需要下载后再次重命名。
- 支持视频标题、分集信息、AV/BV/CID/SS/EP、UP 主、发布时间以及实际画质、分辨率、视频和音频规格等中文字段。
- 每集下载前按最终选流渲染路径；发生规格回退时，文件名会反映实际下载结果。
- 子文件夹留空时不创建额外目录，同名文件自动追加 `(2)`、`(3)`，不会覆盖已有内容。
- 下载历史会保存当次命名规则和每集最终相对路径，加载配置或重试失败集不受之后的全局规则修改影响。

## 历史记录

- 下载记录与重命名记录整合到同一个“历史记录”页面，通过标签页切换。
- 批量下载记录显示最终视频、音频规格，并可进入详情查看每集结果。
- 规格标签支持自适应宽度和换行，长规格不再被截断。
- 历史列表填满窗口剩余高度，记录间增加浅色分割线；任务类型和时间在每行垂直居中。

## 界面与兼容性

- “命名模板”更名为“命名规则”，原有影视重命名模板继续在独立标签中管理。
- 下载设置区域显示当前自动应用的命名规则和路径示例，并提供管理入口。
- `AppSettings` 升级到 schema 4；旧配置会自动补充兼容的默认下载命名规则，不改变原有下载目录和其他设置。
- 更新 README 截图，并修正导航、字段卡片宽度和多处布局细节。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
