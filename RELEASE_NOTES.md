# Beflow for Windows v1.0.4

本次版本集中修复历史记录并发、多音轨完整性、页面生命周期和便携更新可靠性问题。

## 修复与改进

- 历史记录增加稳定 ID，删除操作不再因列表刷新或新任务插入而误删其他记录。
- 配置与历史 JSON 改为串行原子更新，避免同时保存时发生临时文件冲突或记录丢失。
- “补全旧记录标题”改为按记录 ID 合并，不会覆盖刚完成下载产生的新历史。
- 多音轨封装会检查缺集、重复分P、无效 `[Pxx]` 前缀和奇偶数量不一致，不再静默输出不完整结果。
- 运行任务时关闭窗口会先显示确认对话框，避免误操作直接取消下载或封装。
- 下载、多音轨和设置页面会在导航离开时解除任务状态订阅，避免长期使用后的页面与 ViewModel 累积。
- 便携更新使用内部文件清单删除旧版遗留组件，同时保留 `Data`、`portable.flag` 和非程序用户文件。
- 更新失败时会恢复被覆盖及删除的文件；如果回滚自身失败，会保留备份目录并写入更新日志。
- 在线更新、关于页、安装包和便携包现已支持四段版本号。

安装版覆盖升级和便携版更新都会保留现有配置、主题选择、历史、任务日志及 B 站登录数据。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
