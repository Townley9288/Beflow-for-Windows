# Beflow for Windows v1.1.0.2

本次版本改进 BBDown 链接解析失败时的诊断体验，解决界面只显示“退出码 1”、无法获得实际失败原因的问题。

## 解析日志

- “解析当前集”和“解析全部集”现在会遵循“保存任务日志”设置，将 BBDown 解析过程写入应用日志目录。
- 安装版日志位于 `%LOCALAPPDATA%\Beflow\Logs`；便携版日志位于程序旁的 `Data\Logs`。
- 解析失败提示新增“打开日志”按钮，可直接查看本次任务日志，无需手动寻找文件。
- 解析日志继续使用 30 天、最多 100 份的现有清理规则。

## 错误提示与隐私

- BBDown 返回非零退出码时，提示会附带输出末尾的有效错误内容，不再只显示退出码。
- 自动忽略 BBDown 版本横幅、项目提示、任务完成文字和临时媒体地址，只保留有助于定位问题的内容。
- 错误提示和日志统一隐藏 Cookie、SESSDATA、CSRF、用户标识、access token 与 refresh token。
- `bilivideo.com` 临时视频或音频地址不会写入日志，也不会出现在错误提示中。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
