# Beflow for Windows v1.0.2

这是一次日志、历史记录和更新界面体验修复。

## 修复与改进

- 下载页与多音轨页的运行日志会自动跟随最新输出；用户主动向上滚动时暂停跟随，开始新任务时自动恢复。
- 下载完成后直接使用本次 BBDown 任务已经解析到的标题写入历史，不再产生空标题或额外重复请求。
- 多音轨下载完成后同样自动保存主版本标题。
- 历史页在任务完成并写入记录后自动刷新，无需手动点击“刷新”。
- “刷新标题”改为“补全旧记录标题”，仅用于修复旧版本已经产生的空标题记录。
- 历史时间改为本地 `yyyy-MM-dd HH:mm:ss` 格式，不再显示容易混淆的时区偏移后缀。
- 关于页更新日志支持完整换行与滚动查看，不再被固定高度裁掉。
- 更新日志中的 Markdown 标题和列表符号会转换为更适合原生界面的纯文本格式。

安装版覆盖升级和便携版更新都会保留现有配置、历史、任务日志及 B 站登录数据。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
