# Beflow for Windows v1.1.0.4

本次版本将多音轨功能升级为“先解析、后选流、再下载封装”，并修复下载历史恢复、影视重命名和 TMDB 设置中的多项状态与并发问题。

## 多音轨解析式下载

- 双链接与同一链接奇偶分P均可解析当前集或全部集，双方分集默认按顺序配对，并支持逐集调整来源 B 的对应分集。
- 来源 A、B 可分别选择画质、编码与音频规格；主视频支持智能推荐、固定来源 A 或固定来源 B，并显示推荐理由。
- 下载前会重新确认双方实际流规格，逐集下载主视频、A/B 音轨并使用 mkvmerge 封装；单集失败不会中断后续队列。
- 音轨名称、语言、默认音轨和来源 B 相对来源 A 的延迟始终按来源绑定，不会因主视频来源改变而错位。
- 新任务保存 `dual-audio-task.json` 清单，可选择保留中间文件，并支持从清单或旧“主版本/副音轨”目录重新封装。
- 新增总体与当前分集进度、失败来源单独重试、多音轨历史详情、恢复计划和重新封装入口。

## 稳定性与历史恢复

- 下载链接改变后会立即使旧解析结果失效，避免误用上一个链接的分集和规格。
- 从历史恢复手动指定的视频或音频规格时进行完整签名匹配；规格已经失效会明确提示，不再静默改选相近码率。
- 修复重命名预览重复启动、历史搜索结果乱序和活动模板快速切换可能造成的竞态。
- TMDB Key 现在直接验证页面中的未保存内容；验证失败不会保存无效 Key，保存 TMDB 设置也不会覆盖模板和当前模板选择。
- 模板保存与另存前会拒绝未知字段，TMDB 搜索窗口可随应用宽度自适应。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
