# Beflow for Windows v1.1.0.0

本次版本重构普通下载流程，改为“先解析、后选流、再下载”，并补齐逐集规格控制、批量进度、历史详情以及两种下载器的实时速度显示。

## 新功能

- 支持“解析当前集”和“解析全部集”；整季解析会增量显示结果，可随时取消并保留已完成分集。
- 新增全宽分集列表，展示每集画质、真实分辨率、码率、编码、音频规格和预估大小。
- 支持顶部批量选流规则，以及每一集独立覆盖画质、编码和音频；切换画质后编码会自动匹配实际可用规格。
- 下载前会重新解析当前分集并按规格签名匹配最新流序号，避免复用失效的临时索引。
- 多集任务逐集顺序下载，单集失败后继续下一集，并显示总体进度、当前集进度、速度和剩余时间。
- 新增批量下载历史详情，保存每集最终规格、状态、错误和输出文件，并支持加载配置及重试失败集。

## 下载与性能

- aria2c 可直接显示百分比、速度和剩余时间，并根据单个媒体流大小自动使用 4、8 或 16 连接；手动参数作为上限，也可在设置中关闭自动调优。
- aria2c 在 Windows 上关闭文件预分配并启用 64MB 磁盘缓存，减少启动和写盘开销。
- 关闭 aria2c 时改用 Windows ConPTY 读取 BBDown 内置下载器的终端进度，可显示百分比、速度和估算剩余时间。
- 视频与音频进度会按解析到的实际大小合并计算，切换音频阶段时总体进度不会回退。
- 取消任务只终止本次会话启动的进程树，不扫描系统进程，也不删除其他任务文件。

## 界面与设置

- 下载页改为自适应单列布局，支持分集搜索、全选、反选、动态大小统计以及可展开的下载配置。
- 下载完成提示中的“打开下载文件夹”和“去重命名”调整为同排操作。
- 历史列表增加任务规格摘要，批量任务可进入独立详情页查看每集结果。
- 设置页各功能卡片独立保存，软件更新保持在页面底部；aria2c 卡片新增自动调优开关和上限说明。
- 默认窗口和主要页面进一步适配 4:3 布局，并保持不同窗口尺寸下的响应式排列。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志和 TMDB 设置。现有配置缺少新字段时会自动使用兼容默认值。
