# Beflow for Windows v1.1.0.6

本次版本优化影视重命名页面的长列表布局，并将重命名历史详情升级为完整的结构化信息窗口。

## 重命名页面布局

- 重命名历史、撤销、取消任务、预览和执行按钮固定在页面底部，不再因文件数量或预览内容增加而滚出窗口。
- 源文件与重命名预览列表占用剩余可用空间，内容过多时分别在列表内部滚动。
- 窄窗口下两个列表平均分配可用高度；窗口高度不足时配置区域单独滚动，并优先为列表及底部操作栏保留空间。

## 重命名历史详情

- “查看详情”升级为宽版结构化窗口，集中显示中英文片名、媒体类型、年份、季数、命名模板、执行状态以及执行和撤销时间。
- 增加来源文件夹、总变更数量、媒体文件与字幕等关联文件分类统计。
- 每项文件名变更分别显示原文件名、新文件名、序号和文件类型；长记录在窗口内滚动，完整路径可通过悬停查看。
- 中文片名与原始片名相同时显示“同中文片名”，避免重复占用信息区域。

安装版覆盖升级和便携版更新都会保留现有配置、主题、账号登录数据、下载与重命名历史、任务日志、TMDB 设置和自定义规则。

发行文件暂未进行商业代码签名，Windows SmartScreen 可能显示未知发布者。请从本 Release 下载，并使用对应 `.sha256` 文件核验完整性。
